    Doing Stuff with Web Things

    One thing the I learned as I read this section was that when you want to get the user’s input from any input or text area tag, you use the special word 'value'.

    For example:

    let namevar = document.getElementById(“name_input”).value;

    You are storing the value in the 'namevar' variable from the user.

    When you want to show something to a user, you use a section and the special word ‘innerHTM’. For example:

    document.getElementById(“display”).innerHTML = theinformation

    You are showing the information which is stored in the 'theinformation' variable to the user. This infomation or data appears in the place where 'display' id is located.
